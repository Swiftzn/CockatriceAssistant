## üîß Troubleshooting

**App won't start?**
- Check Windows antivirus (some flag PyInstaller files)  
- Try running as administrator
- Ensure you have internet connectivity

**Decks not appearing?**
- Verify Cockatrice folder detection worked
- Try manual folder selection
- Restart Cockatrice after importing

**Theme installation failed?**
- Close Cockatrice before installing themes
- Check folder permissions
- Try running as administrator
